## Summary

Most prior systems rely on accelerometer- or airbag-triggered detection combined with GPS/GSM for notification, with more recent work exploring smartphone sensors, camera/face recognition, and machine learning for severity prediction. This project differentiates itself by using a low-cost vibration-sensor-based approach that does not depend on airbag deployment, making it adaptable across a wider range of vehicle types.
